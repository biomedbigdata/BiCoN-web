Data format

The algorithm needs as an input one CSV matrix with gene expression/methylation/any other numerical data and one CSV file with a network.

Numerical data

Numerical data is accepted in the following format:

  • genes as rows.
  • patients as columns.
  • first column - genes IDs (can be any IDs).

Network

Even though we support our users with already processed PPI networks, it is still possible to load a user-defined network. An interaction network should be present as a CSV table with two columns that represent two interacting genes. Without a header!

Data processing

For internal calculations, BiCoN normalizes the data by applying log2 transformation and then applying z-scores normalization. If your data was already log2 scaled, please remove a check from "Log2 transform".

Algorithm parameters

The main parameters of the algorithm are the sizes of the desired solution. Please, indicate what is a minimal and maximal number of genes you would like to have in each subnetwork.

The algorithm works such that in most of the cases default parameters deliver the optimal performance. If you want to set advanced parameters, please check "Use advanced parameters: Yes". A user can specify the following parameters:

  • Gene set size - number of genes considered in the analysis. The algorithm usually preselects 3000 the most variant genes to speed-up the calculation. This is usually an optimal choice as genes which can be selected for clustering, are supposed to have a high variance. Therefore, if needed this parameter can be increased, but we do not recommend to go higher than 5000 to guarantee a reasonable run-tyme
  • Maximum number of iterations - we generally do not recommend to terminate the algorithm before convergence which is usually reached in 30-60 iterations) and therefore we remend to change this parameter only for testing purposes
  • Number of ants - more ants can make more search space exploration, but also increase the run-time
  • Evaporation rate - higher evaporation rate will speed-up convergence, but also increase risks to get stuck in a local minima
  • Pheromone significance and Heuristic Information significance please modify it only if you really well aware of the role of those parameters in Ant Colony Optimization algorithms convergence.

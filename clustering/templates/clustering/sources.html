Vizualise Gene Expression and PPI data.

Sources used for this web application:

Sources

The two built-in expression datasets are from the following sources:
Breast Cancer
Lung Cancer

The built-in PPI data are loaded from the NDEx Bio webserver and contain datasets from the following sources: Human Protein Reference Database (link to source), Agile Protein Interactions Database (link to source), STRING database (link to source), and from BioGrid (link to source).

Most of the code for this website is written in Python.
As a server framework, Django Version 2.1.3 is used.
The queuing system Celery is used, based on a RabbitMQ server for request processing.
For the frontend, additional JavaScript and JQuery code is used.
For the PPI network display, the javascript library Sigma.js is used.
The javascript library DataTables is used to improve table display.

The server uses the following python libraries:
django-livereload-server
Matplotlib for plots,
Plotly for plots,
Numpy,
Scipy,
Bokeh for graph preprocessing,
NetworkX for graph processing,
Seaborn for the heatmap,
PyBiomart for gene name conversions,
NDEx 2 for PPI data import,
Pandas for internal data processing,
GSEApy is used for gene set enrichment analyses,
Scikit / sklearn for clustering,
Lifelines for survival analysis
For enrichment analyses, the datasets (that are built-in in gseapy) from the following sources are used: KEGG, GO (Gene Ontology): Biological Process and Molecular Function, and Reactome Pathway Database.
